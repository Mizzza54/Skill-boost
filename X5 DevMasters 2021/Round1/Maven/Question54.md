# Вопрос 54
_Какой командой maven можно получить список всех зависимостей проекта:_

## Варианты ответов:

- `mvn -dependency:tree`
- `mvn -dependency:list`
- `mvn -dependency:show`
- `mvn -dependency:showtree`
- `mvn -showdependency`
- `mvn dependency:tree`
- `mvn dependency:list`
